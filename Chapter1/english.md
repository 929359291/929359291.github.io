#网易云大学自学英语资料片段

```
阅读非小说类作品至关重要
Reading non-fiction writing critically involves ( B ).
A.
supporting the author’s opinion 支持作者的意见

B.
developing your own opinions 发展自己的意见

C.
understanding what is written 理解写的是什么

D.
reading without any imagination 阅读没有假想
```


```
在接受文章中的内容之前，你应该确定
Before accepting what is written in an article, you should be sure that ( A ). 
A.
enough support for the assertions has been given 已经给出了对断言的足够支持

B.
your values and attitudes have been reflected in it 你的价值观和态度已经反映在其中

C.
it has been checked from an authoritative source 它已从权威来源检查过

D.
you have found the differences in the writing styles 你发现了写作风格的差异
```

```
如果文章中有一些不一致的点，你应该这样做
If there are some inconsistent points in the article, you should ( D ).
A.
insist on your own opinion and confirm it 坚持自己的意见并确认

B.
avoid mentioning the inconsistencies 避免提及不一致

C.
emphasize the work of authorities 强调当局的工作

D.
examine the author’s supportive statements 检查作者的支持性陈述
```

```
爱因斯坦是这篇文章的相关来源
Einstein is a relevant source when the article is about ( A ).
A.
physical breakthroughs  物理突破

B.
the subject of poetry  诗的主题

C.
studies in current physics 当代物理学研究

D.
credible writing sources 可靠的书面资料
```
